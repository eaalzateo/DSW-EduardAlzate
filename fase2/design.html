Propósito del sitio

El proposito del sitio web es diseñar 5 objetos virtuales de aprendizaje de los temas de Diseño de Sitios Web. Que permitan suplir la necesidad de la presencialidad y apoye la educación virtual, para el aprendizaje.

Etiquetas semánticas, listas y enlaces

Etiquetas semánticas

Las etiquetas semánticas, son aquellas que dan un significado al documento y nos permiten crear un maquetado del diseño web. Ayudan a definir la estructura del documento y permiten que las páginas sean mejor indexadas por los buscadores web.

Bibliografía

Celaya Luna, A. (2014). Creación de páginas web: HTML 5. Recuperado de: https://elibro-net.bibliotecavirtual.unad.edu.co/es/ereader/unad/56045

Listas

Las listas, son aquellas etiquetas que permiten crear conjuntos de elementos en forma de listas; estas pueden ser ordenadas o desordenadas. Normalmente, se encuentran precedidas de un guión o número.

Una lista ordenada, se representa mediante la etiqueta ol; y cada uno de sus elementos, se representa mediante la etiqueta li

Ejemplo de lista ordenada:

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Una lista desordenada, se representa mediante la etiqueta ul; y cada uno de sus elementos, se representa mediante la etiqueta li

Ejemplo de lista desordenada:

  • Elemento 1
  • Elemento 2
  • Elemento 3

Bibliografía

Celaya Luna, A. (2014). Creación de páginas web: HTML 5. Recuperado de: https://elibro-net.bibliotecavirtual.unad.edu.co/es/ereader/unad/56045
Definición listas. Manual web. Etiquetas de maquetación, ul, ol, li. Recuperado de: https://manualweb.net/

Enlaces

Los enlaces, son aquellos que nos sirven para enlazar contra otro recurso de la web. Estos se emplean con la etiqueta a y su atributo principal es href, en el cual, se encuentra la ubicación del destino, el cual deseamos enlazar.

Ejemplo de enlace:

Ir a google.com
